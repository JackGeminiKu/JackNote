定义

命名函数


// 无参函数	
function fn1(){
    alert("Hello World!");
}

// 带参函数
function bark ( dogName, dogWeight ) {
    if ( dogWeight <= 10 ) {
        return dogName + " says Yip";    // 函数返回值
    } else {
        return dogName + " says Woof";
    }
}

// 定义函数变量
var fn = Fn1;

匿名函数


var Fn2 = function(){    // 符号"="后面的是匿名函数
    alert("Hello wild!");
}

函数覆盖: 使用多个外部JavaScript文件时有一点要记住, 如果不同文件中有两个同名的函数, 后面的函数将覆盖前面函数的定义!

调用函数

bark("Tom", 30);    // dogName = "Tom"; dogWeight = 30;

匿名函数 + 立即调用


(function(x){
    alert(2* x); 
})(3);

在一个块(文件)中，函数可以先调用, 后进行“定义式”定义；但在两个块中，必须先定义, 后调用.
