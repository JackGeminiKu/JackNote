画布(canvas)介绍
画布颜色

画布是透明的. "画笔"默认颜色是黑色.

画布宽度 & 高度

默认的, 画布元素是300像素宽, 150像素高. 如果未在canvas标记中指定width和height属性, 就会得到这个大小的画布. 如果接下来在CSS中指定了一个大小, 比如600px x 200px, 那么原来的300 x 150的画布就会扩展到这个大小, 画布中绘制的所有内容也会随之扩展. 这就像为一个图像指定新的宽度和高度时, 如果比原图像的实际宽度和高度更大或更小时, 就会将图像拉伸或缩小. 不过, 如果使用元素的width和height属性, 会把画布的大小设置得比300 x 150更大(或更小), 但画布上的所有内容仍然会正常绘制. 所以我们建议还是在标记属性中指定宽度和高度, 而不要在CSS中设置这些属性, 除非你本来就打算缩放画布.

如果你只想检查是否支持画布, 而无需标记中有一个画布, 那么:


function canvasSupport() {
    return !!document.createElement('canvas').getContext;
}
画矩形

context.fillType = "lightblue";    // 指定填充色
context.fillRect(10, 10, 100, 100);    // 参数分别是: x, y, width, height, 但是是像素
context.strokeRect(10, 10, 100, 100);    // 画矩形轮廓
画三角形

context.beginPath();    // 开始绘制path
context.moveTo(100, 150);    // 把"铅笔"移动到开始位置(100, 150)
context.lineTo(250, 75);    // 画一条线到(250, 75)
context.lineTo(125, 30);    // 画一条线到(125, 30)
context.closePath();    // 最后画一条线到开始位置
context.lineWidth = 5;    // 设置线宽
context.stroke();   // 用线描路径
context.fillStyle = "red";    // 设置笔的颜色为红色 
context.fill();    // 用红色填充三角形
画圆(弧)

context.beginPath();
context.arc(150, 150, 50, 0, 2 *Math.PI, true);
context.fillStyle = "lightblue";
context.fill();
写字

// 设置对齐, 可设置: left, right, center, start & end
context.textAlign = "left"; 
// 设置字体
context.font = "2em Lucida Grande";
// 设置颜色
context.fillStyle = "red";
// 设置基线, 可设置: alphabetic, bottom, middle & top
context.textBaseline = "middle";
// 填充or笔划
context.fillText("Dog", x, y, max_width);
context.strokeText("Dog", x, y, max_width);
