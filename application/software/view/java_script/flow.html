脚本使用位置
在head元素中的Script标签: 在页面加载之前就执行了.

Scripts in the header can’t create output within the HTML document, but can be referred to by other scripts. The header is often used for functions.

在body元素中的Script标签: 读到即执行，写在事件触发里除外。

In this case, the script’s output is displayed as part of the HTML document when the browser loads the page.

在HTML tag中: 事件处理

This is called an event handler and allows the script to work with HTML elements. When using JavaScript in Event handlers, you don’t need to use the <script> tag. 


<img SRC="button.gif" onMouseOver="highlight();">

补充一下, 还有两种情况: ① 一种是加了defer属性的在文档加载后执行，可以理解为放在html最底部; ② 另外，html5 增加了 async 属性，相当于异步执行，与文档加载同时进行;
