数据类型 解释 字面值
字符串 VARCHAR(n) 变长字符串, 最大长度只有65535个字符 '我是字符串'
CHAR(n) 定长字符串, 最大长度只有255个字符
TINYBLOB/BLOB/MEDIUMBLOB/LONGBLOG 海量字节字符串
TINYTEXT/TEXT/MEDIUMTEXT/LONGTEXT 海量字节字符串
浮点数 DEC(3,2) decimal的缩写, 表示浮点数. DEC(3,2)表示一共有3位数字, 其中2位是小数 3.14
整数 tinyint 单字节整数, 表示0~255 100
smallint 双字节整数
int 4字节整数
bigint 8字节整数
日期时间 DATE 年-月-日 '1978-9-15'
TIME 时:分:秒 '08:56:00'
DATETIME 年-月-日 时:分:秒 '1978-9-15 08:56:00'

备注:
1) NULL: NULL好比一个还没打开的盒子, 盒子里可能装有任何事物, 也可能是个空盒子. 所以无法比较两个NULL, 因为我们根本不知道盒子里有什么!
2) 单引号: 单引号是SQL中的保留字符, 字符串中的单引号要用\'来表示. '\'要用'\\'来表示, '>'要用'\>'来表示!
