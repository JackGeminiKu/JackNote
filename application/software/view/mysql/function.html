字符串函数
RIGHT()和LEFT() 从列中选出指定数量的字符. 例如: SELECT RIGHT(location, 2) FROM my_contacts; # 从右侧开始, 选取两个字符串
SUBSTRING() 返回指定位置的字符串. 例如: SELECT SUBSTRING('San Antonio, TX', 5, 3);
SUBSTRING_INDEX() 返回指定字符串前的所有内容. 例如: SELECT SUBSTRING_INDEX(location, ',', 1) FROM my_contacts; # 找到第一','符号, 返回这个符号前的所有字符

分组函数
SUM 对组中某一列数据求和. SELECT first_name, SUM(sales) FROM cookie_sales GROUP BY first_name ORDER BY SUM(sales) DESC;
