逻辑运算

AND, OR

比较运算

=, <, >, <=, >=

字符串也支持比较运算

LIKE (NOT LIKE)

location LIKE '%CA';
NOT location LIKE 'A_';

备注:
1) 通配符： ① %是>=0个未知字符的替身； ② _表示1个未知字符的替身.
2) like不区分大小写

IS NULL (IS NOT NUL)

calories IS NULL;
calories IS NOT NULL;
BETWEEN AND

calories BETWEEN 30 AND 60    # 范围包括30和60
NOT calories BETWEEN 3 AND 5
IN (NOT IN)

rating IN ('innovative', 'fabulous', 'delightful', 'pretty good');
rating NOT IN ('innovative', 'fabulous', 'delightful', 'pretty good');
CASE表达式

CASE
    WHEN drama = 'T' THEN 'drama'
    WHEN comedy = 'T' THEN 'comedy'
    ELSE 'misc'
END;
