1. 算术运算
　 加, 减, 乘, 除, 求余 +, -, *, /, % 　
　 前自加(减) ++i; --i 前自增(减): 首先i自增, 表达式的值等于自增之后的值.
　 后自加(减) i++; i-- 后自增(减): 表达式的值等于自增之前的值, 然后i自增
2. 比较运算
　 小于(等于)  <, <= 　
　 大于(等于)  >, >= 　
　 等于  == 比较primitive类型的值是否相等, 或者引用类的值是否引用的同一对象
　 不等于 != 　
3. 逻辑运算
　 逻辑非 ! 　
　 逻辑或 ^ 　
　 短路或 || 短路求值
　 短路与 && 短路求值
4. 位运算
　 位与, 位或, 位异或, 位取反 &, |, ^, ~ 位与
　 位移 >>, << 位右移, 位左移
　 　 >>> 　
5. 赋值运算
　 = 　 　
　 +=, -=, *=, /=, %= 　 　
6. 字符串连接
　 + 　 "+"运算符两侧的操作数中, 只要有一个是字符串(String)类型, 系统会自动将另一个操作数转换为字符串然后再进行连接!
7. 三元运算
　 x ? y : z 　 　
8. instanceof
　 garfield instanceof Cat 　 如果garfield是Cat, 则返回true, 否则, 返回false
