Java共有4种类型的整数: byte, short, int和long, 代表不同大小的整数. 4种类型的整数都是primitive类型.

类型 占用字节 表示的数值范围 字面值
byte 1 -128 ~ 127 123
short 2 -65536 ~ 65535 123
int 4 -231 ~ 231-1 123(不加后缀, 默认是int类型)
long 8 -263 ~ 263-1 123L

备注: Java没有无符号的整数, 所有整数都是有符号的!

各种进制

10进制 123
8进制 0123(以数字0开头)
16进制 0x123
