Java只能单继承, 但是可以实现多个接口!

定义接口

Java中的接口是"抽象方法"和常量值的定义的集合!


interface Pet {
    // 常量
    [ public static final ] int id = 1;

    // 抽象方法
    [ public abstract ] void play( );
}

注意: []中的写和不写效果是一样的!

实现接口

class Dog implements Pet {  // 可以实现多个接口
    ...
}
继承接口

interface A extends B {
   // 一个接口可以继承自另一个接口, 添加其他常量&方法
}
接口 vs 多态

接口也支持多态. 而且接口作为多态类型时, 对象可以是任何对象, 而类作为多态类型时, 对象必须是子类型!!!
