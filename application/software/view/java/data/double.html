Java种有两种类型的浮点数: float & double, 表示不同范围的浮点数(浮点数有误差).

类型 占用字节 表数范围 字面值
float 4字节, 范围规模可变 -3.403E38 ~ 3.403E38 3.14F
double 8字节, 范围规模可变 -1.798E308 ~ 1.798E308 3.14 (不加好后缀, 默认是double型)

注: 浮点数是有误差的!

十进制 & 科学技术

十进制形式, 例如: 3.14
科学计数法形式, 例如: 3.14e2

注:  Java浮点型常量默认为double型, 如要声明一个常量为float型, 则需在数字后面加上"F", 如: 3.14F.
