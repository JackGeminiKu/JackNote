  1. 条件

    • 单路if语句

      if(条件成立){
          ...
      }

      双路if语句

      if(条件成立){
          ...
      }else{
          ...
      }

      　

      也可使用条件运算

      多路if语句

      if (条件1成立){
          ...
      }else if (条件2成立){
         ...
      }else if (条件3成立){
          ...
      }else{
          ...
      }

      switch语句

      switch (<testVar>){  // <testVar>必须是整数类型, <comparsionValX>必须是一个常量, 不能是变量或表达式
          case <comparisonVal1>:
              // 当testVar的值为comparisonVal1时
              break;
          case <comparisonVal2>:
          case <comparisonVal3>:
              // 当testVar的值为comparisonVal2或comparisonVal3时
              break;
          default:
              // 其他情况
              break;
      }
  2. 循环

    • for循环

      for (初始化; 条件; 更新){
          // code to be looped
      }

      while循环

      while(条件){
         // code to be looped
      }

      do循环

      do{
         // code to be looped
      }while(条件);

      加强版for循环

      for ( String item : array ) {
          // 遍历array数组
      }
  3. 跳转

    • break;

      立即跳出循环或select语句.

      continue;

      立即终止当前的循环, 继续执行下一次循环.

      return;

      跳出函数.
      register_shutdown_function: 注册一个 callback ，它会在脚本执行完成或者 exit() 后被调用。
  4. 注释
    • // 单行注释
    • /*
         多行注释
      */
    • /**
         Java doc文档注释
      */
