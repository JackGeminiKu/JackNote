整数分类
类型 含义 字面值
sbyte 单字节, 有符号整数, 表示范围: -128 ~ 127 100
byte 单字节, 无符号整数, 表示范围: 0 ~ 255 100
short 双字节, 有符号整数, 表示范围: -32768 ~ 32767 100
ushort 双字节, 无符号整数, 表示范围: 0 ~ 65535 100
int 4字节, 有符号整数, 表示范围: -21亿多 ~ 21亿多 100(默认)
uint 4字节, 无符号整数, 表示范围: 0 ~ 42亿多 100U
long 8字节, 有符号整数 100L
ulong 8字节, 无符号整数 100UL
16进制表示

前面添加"0x", 例如: 0xFF.

数据转换

Convert.ToInt32(); // 将指定的值转换为int
int.Parse(); & int.TryParse();
255.ToString("X4"); // 结果为: "00FF"

格式化

255.ToString("X4"); // 结果为: "00FF"
