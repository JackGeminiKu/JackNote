  • 匿名委托
  • Lambda表达式 - 简化的匿名委托

    • Lambda运算符"=>", 可以读作"转到"或者"成为. 运算符将表达式分为两部分, 左边指定为输入参数, 右边是Lambda的主体.

    一个参数: param => expr

    Func <int, string> f = p => p.ToString();

    多个参数: (param-list) => expr

    Func <int, int, int> f = (p,z) => p + z;

    主体运算比较复杂

    Func <int, int, int> f = (p,z) => { if (p > z) { return p; } else { return z; } }
