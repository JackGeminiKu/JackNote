元字符
元字符 含义
. 匹配任意字符, 除了换行符
\w 匹配字母, 数字, 下划线, 或汉字
\d 匹配一个数字, 0~9
\s 匹配任意一个空白符
^ 匹配字符串的开始
$ 匹配字符串的结束
\b 匹配单词的开始或结束
转义字符

如果你想查找元字符本身的话, 比如你查找".", 或者"*", 需要使用转义字符!

字符 转义字符
. \.
* \*
\ \\
| \|

) \)

} \}
重复描述符
重复描述符 含义
? 有或者无
* >=0次
+ >=1次
{n} 重复n次
{n,m} n~m次
{n,} >=n次
枚举
枚举 含义
[0~9] 匹配一个数字: 0~9
[abc] 匹配一个字符: a,b或者c
[^aeiou] 匹配一个字符, 除了aeiou
分支匹配
| 正则或
子表达式

使用括号表示子表达式, 例如: (表达式1)(表达式2)

例子
string temp = "ErrorCode:-1,Message:{\"UserId\" : \"1000\",\"userName\" : \"ZhangSan\"}";
Regex reg = new Regex("ErrorCode:(?<key1>.*?),Message:{(?<key2>.*?)}");
Match match = reg.Match(temp);
string tempStr = match.Groups["key1"].Value + "--" + match.Groups["key2"].Value;
　
MessageBox.Show(tempStr);
Match match = Regex.Match("age=30", @"^(.+)=(.+)$");
if (match.Success){     
    Console.WriteLine(match.Groups[0] .Value);//输出匹配的子字符串
    Console.WriteLine(match.Groups[1] .Value);//获取第一个分组的内容
    Console.WriteLine(match.Groups[2] .Value);//获取第二个分组的内容
}
