Nullable<T>表示可分配null的值类型.T 可以是包括 struct 在内的任何值类型；但不能是引用类型。

定义变量

Nullable<int> nullableInt;
int? nullableInt;    // 简写
常用操作
Value

如果变量为null, 访问Value属性会抛出System.InvalidOperationException类型的异常.

.HasValue

如果HasValue是false, 就说明变量被赋予了null.

表达式

通常可空类型与一元或二元元算符一起使用时, 如果其中一个操作数或两个操作数都是null, 其结果就是null. 但是, 在比较可空类型时, 只要有一个操作数是null, 比较的结果就是false.

??运算

当运算式中的一个或两个的值是null时, 对于除了bool?之外的所有简单可空类型, 该操作的结果是null, 可以把它理解为"不能计算".
实际编程时建议先手动计算一下各种情况, 或者使用??运算符.
int? op1 = null;
int result = op1 * 2 ?? 5; // 因为op1是null, 所以op1*2也是null. 但是, ??运算符检测到这个情况, 并把值5赋予result.

op_Implicit

T值可以隐式转换为Nullable, 但是Nullable不能隐式转换为T.

op_Explicit

显式转换.
