Array是一个抽象类, C#中的数组都是它的子类.

普通数组
定义

int[] myIntArray;	// 定义一个整型数组变量, 还未指向具体的整型数组 
myIntArray = new int[5];	// 指向一个整型数组, 维数为5.

int[] myIntArray = new int[5];	// 定义 + 初始化: 所有的数组元素都被赋予同一默认值, 对于数值类型来说, 其默认值是0. 数组维数可以是常量, 甚至是变量!!!

int[] myIntArray = new int[5] {1,2,3,4,5};	// 定义 + 初始化: 指定了所有的元素的初始值

Array arr = Array.CreateInstance(elementType, count);	// 动态创建
操作
访问数组元素

array[i]

获取数组长度

array.Length

颠倒

Array.Reverse(array)

运行时

Array values = Array.CreateInstance(typeof(string), 3);
values.GetValue(0);
values.SetValue("new value", 0);
矩形数组(多维数组)
定义

double[,] hillHeight = new double[3,4];
double[,] hillHeight = { {1,2,3,4}, {2,3,4,5}, {3,4,5,6} };
操作

访问元素


hillHeight[2,1]
锯齿数组(数组的数组)
定义

多维数组可称为矩形数组, 这是因为每一行的元素个数都相同. 而数组的数组, 也可以使用变长数组, 其中每行可以有不同的元素个数. 为此, 需要有这样一个数组, 其中的每个元素都是另一个数组. 也可以有数组的数组的数组, 甚至更复杂的数组. 但是注意这些数组都必须有相同的基本类型.

先定义


int[][] jaggedIntArray;    // 数组中的每个元素也是数组

然后初始化


jaggedIntArray = new int[2][];
jaggedIntArray[0] = new int[3];
jaggedIntArray[1] = new int[4];

jaggedIntArray = new int[2][] = { new int[] {1,2,3}, new int[] {1,2,3,4} };

定义+初始化


int[][] jaggedIntArray = { new int[] {1,2,3}, new int[] {1,2,3,4} };
操作

jaggedIntArray = new int[2][];
jaggedIntArray[0] = new int[3];
jaggedIntArray[1] = new int[4];

jaggedIntArray = new int[2][] = { new int[] {1,2,3}, new int[] {1,2,3,4} };
父类数组 & 子类数组

Animal[] animals;
Dog[] dogs = ...;
animals = dogs;    // 父类数组可以"指向"子类数组
