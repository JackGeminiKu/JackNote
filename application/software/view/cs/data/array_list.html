Implements the IList interface using an array whose size is dynamically increased as required.

ArrayList在使用值类型数据时, 会不断地做装箱和拆箱的工作, 这显然是非常糟糕的事情.

定义

ArrayList animalArrayList = new ArrayList();
操作

添加元素


Add(Object);
AddRange(ICollection);
InsertRange(index,ICollection);

删除元素


Remove( );   
RemoveAt( );

注: 删除当前索引的项, 会使后面的项前移一个位置.

访问元素


arrayList[0];    // 由于ArrayList是Object对象的集合, 所以可能需要类型转换.

查找


IndexOf( );

获取长度


arrayList.Count;

排序


arrayList.Sort();	// 使用每个元素的 IComparable 实现对整个 ArrayList 中的元素进行排序。
aarrayList.Sort(IComparer);    // 使用指定的比较器对整个 ArrayList 中的元素进行排序。
arrayList.Sort(Int32, Int32, IComparer);    // 使用指定的比较器对 ArrayList 中某个范围内的元素进行排序。
