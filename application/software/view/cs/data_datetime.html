DateTime
常用操作
转换成字符串

.ToString() '// 该结果取决于计算机上的区域设置. yyyy: 年; MM: 月; dd: 日; HH: 24小时制; hh: 12小时制; mm: 分; ss: 秒; fff: 毫秒
.ToLongDateString()
.ToShortDateString()
.ToLongTimeString()
.ToShortTimeString()
.ToString("dddd") '// 工作日名称, 由区域设置决定
.ToString("MMMMM") '// 月份名称, 由区域设置决定

提取日期属性

.Year .Month .Day .Hour .Minute .Second .DayOfWeek '// 不能确保Day 1总是指星期一, 同区域设置有关
.DayOfYear

修改日期

.AddDays()
.AddMonths()
.AddYears()

大小比较
>, >=, <, <=, =, !=
计算时间差

(DateTime.Now - previousTestDate).TotalMinutes // -运算的结果是TimeSpan类型

TimeSpan (时间片)

TimeSpan represents a time interval.

定义
TimeSpan timeSpan = new TimeSpan(0,1,0);        // 定义一个时间片: 1分钟
TimeSpan timeSpan = TimeSpan.FromMinutes(1);    // 定义一个时间片: 1分钟
TimeSpan timeSpan = dateTime2 - dateTime1;
