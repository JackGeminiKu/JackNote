安装相关软件

安装Python

Python有32位和64位两种!

安装pip

pip是Python官方推荐的包管理工具.

安装pip: Python get-pip.py (get-pip.py需要下载, Python 3.4已上自带pip, 无需安装)

通过pip安装包
1) pip install scrapy
2) 安装本地wheel文件: pip install wheel文件路径 (各种Python wheel文件)

安装Twisted

安装twisted: pip Twisted-17.9.0-cp36-cp36m-win_amd64.whl (直接安装编译好的, 无需安装几个G的VC 2015)

安装Scrapy

安装scrapy: pip install scrapy

安装PyWin32

To install pywin32 system files, run `python.exe Scripts\pywin32_postinstall.py -install` from an elevated command prompt.

常见错误
私用PIP安装报错: error: Microsoft Visual C++ 14.0 is required. Get it with "Microsoft Visual C++ Build Tools": http://landinghub.visualstudio.com/visual-cpp-build-tools

As the other responses pointed out, one solution is to install Visual Studio 2015. However, it takes a few GBs of disk space. One way around is to install precompiled binaries. The webpage http://www.lfd.uci.edu/~gohlke/pythonlibs (mirror) contains precompiled binaries for many Python packages. After downloading the package of interest to you, you can install it using pip install, e.g. pip install mysqlclient‑1.3.10‑cp35‑cp35m‑win_amd64.whl.
