VIM
各种模式
正常模式

正常模式一般用于浏览文件，也包括一些复制、粘贴、删除等操作。这时击键时，一般的键/键组合会被当成功能键，而不会键入对应的字符。
启动vim后默认位于正常模式。不论位于什么模式，按下(Esc)键(有时需要按两下）都会进入正常模式.

正常模式 ==> 插入模式

在正常模式中按下i, I, a, A, o, O等键，会进入插入模式。
插入模式中，击键时会写入相应的字符。

i(I): 在光标左侧(行首)输入正文
a(A): 在光标右侧(行尾)输入正文
o(O): 在光标所在行下一行(上一行)添加新行

正常模式 ==> 命令模式

在正常模式中，
1) 按下：（冒号）键，会进入命令模式.
2) 按下.符号, 会重复执行上一次的命令.

正常模式 ==> 可视模式

在正常模式按下v, V, (Ctrl) + v，可以进入可视模式。
可视模式中的操作有点像拿鼠标进行操作，选择文本的时候有一种鼠标选择的即视感，有时候会很方便。

常用操作
光标操作
简单方向移动

h: 左移
l: 右移
j: 下移
k: 上移

前面都可以加数字, 代表执行操作的次数.

行移动

gg: 移动到文件头
G: 移动到文件尾
nG: 移动到第n行
H: 移动到当前窗口第一行(head)
M: 移动到当前窗口中间行(middle)
L: 移动到当前窗口最后一行(last)

0: 移动到行首
$: 移动到行尾

ctrl + b: 向上翻页(b为backward)
ctrl + f: 向下翻页(f为forward)

单词移动

w: 移动到下一个单词的首字母
b: 移动到前一个单词的首字母
e: 移动到下一个单词的尾字母

前面可以加数字, 代表执行操作的次数

标记移动

ma: 为当前行设置一个标记, 名字为a. 标记名可以为a~z, A~Z
'a: 返回到标记名为a的那一行
:delmarks a: 删除标记a
:marks: 列出所有的标记

句子

): 下一句
(: 上一句

前面可以加数字, 代表执行操作的次数

光标跳转

ctrl + o: 返回上一次光标的位置
ctrl + i: 同上相反

自动切换输入法

在 Windows 下，用 AutoHotkey 的脚本来处理。
按 ESC 时，将输入法切换为为美式键盘。
脚本如下（保存为 vim.ahk）
~esc::
Send, {ESC}
; 下方代码可只保留一个
SwitchIME(0x04090409) ; 英语(美国) 美式键盘
SwitchIME(0x08040804) ; 中文(中国) 简体中文-美式键盘
return
SwitchIME(dwLayout){
HKL:=DllCall("LoadKeyboardLayout", Str, dwLayout, UInt, 1)
ControlGetFocus,ctl,A
SendMessage,0x50,0,HKL,%ctl%,A
}
发布于 2018-0
