imagecreatetruecolor() 在内存中创建一个空图像, 准备由其他GD函数在它之上进行绘制
imagecolorallocate() 分配一个颜色
imagesetpixel() 绘制一个像素
imageline() 绘制一条线
imagerectangle() 绘制一个矩形
imagefilledrectangle() 绘制一个矩形, 内部填充了颜色
imageellipse() 绘制一个椭圆(或圆)
imagefilledellipse() 绘制一个椭圆(或圆), 内部填充了颜色
imagestring() 绘制一个字符串, 使用PHP内置字体
imagestringup() 同imagestring(), 只是字符串是垂直的
imagettftext() 绘制一个字符串, 可以指定字体
imagepng() 生产图形, 保存在内存或文件中
imagedestroy() 销毁图像相关资源
