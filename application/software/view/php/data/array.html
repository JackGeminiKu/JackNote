PHP可以用 array() 语言结构来新建一个数组。它接受任意数量用逗号分隔的 键（key） => 值（value）对. key 可以是 integer 或者 string。value 可以是任意类型。

定义数组
定义

// 方法1
$productors = array(0 => 'Tires', 1 => 'Oil', 2 => 'Spark Plugs');
$productors = array('Tires', 'Oil', 'Spark Plugs');	// 如果对给出的值没有指定键名，则取当前最大的整数索引值，而新的键名将是该值加一。
$prices = array('Tires' => 100, 'Oil' => 10, 'Spark Plugs' => 4);
$prices = ['Tires' => 100, 'Oil' => 10, 'Spark Plugs' => 4];  // since 5.4
// 方法2
$prices['Tires'] = 100;
$prices['Oil'] = 10;
$prices['Spark Plugs'] = 4;

数组运算
运算符 名称 示例 结果
+ 联合 $arr1 + $arr2 数组$b将被附加到$a中, 但是任何关键字冲突的元素将不会被添加.
== 等于 $arr1 == $arr2 如果$a和$b包含相同的元素, 则返回true; 否则, 返回false.
=== 恒等于 $arr1 === $arr2 如果$a和$b包含相同顺序的相同元素, 则返回true; 否则, 返回false.
!= 不等于 $arr1 != $arr2 如果$a和$b不包含相同的元素, 则返回ture; 否则, 返回true.
!== 不恒等于 $arr1 !== $arr2 如果$a和$b不包含相同顺序的相同元素, 则返回true; 否则, 返回false.
