PHP只有一种整数类型: integer, 它的表示范围同和平台有关:

非windows 32位系统 32位有符号数
非windows 64位系统 64位有符号数
windows 32位系统 32位有符号数
windows 64位系统 PHP 7以前是32位有符号数, PHP 7以后是64位符号数

integer值的字长可以用常量PHP_INT_SIZE来表示
integer最大值可以用常量PHP_INT_MAX来表示(since PHP 4.4.0)
integer最小值可以用常量PHP_INT_MIN来表示(since PHP 7.0.0)

定义整型变量

$a = 1234; // 十进制数
$a = 0123; // 八进制数 (前面是数字0)
$a = 0x1A; // 十六进制数
$a = 0b11111111; // 二进制数字

整数溢出

如果给定的一个数超出了 integer 的范围，将会被解释为 float。同样如果执行的运算结果超出了 integer 范围，也会返回 float。

整除

PHP 中没有整除的运算符。1/2 产生出 float 0.5。 值可以舍弃小数部分，强制转换为 integer，或者使用 round() 函数可以更好地进行四舍五入。

转换为整型

要明确地将一个值转换为 integer:
1) 用 (int) 或 (integer) 强制转换。
2) 不过大多数情况下都不需要强制转换，因为当运算符，函数或流程控制需要一个 integer 参数时，值会自动转换。
3) 还可以通过函数 intval() 来将一个值转换成整型。

将 resource 转换成 integer 时， 结果会是 PHP 运行时为 resource 分配的唯一资源号。

将布尔值转换为integer时, FALSE 将产生出 0（零），TRUE 将产生出 1（壹）。

将浮点数转换为integer时, 将向下取整. 如果浮点数超出了整数范围, 则结果为未定义，因为没有足够的精度给出一个确切的整数结果。在此情况下没有警告，甚至没有任何通知！
