PHP字符串只有一种类型: string.
一个字符串 string 就是由一系列的字符组成，其中每个字符等同于一个字节。这意味着 PHP 只能支持 256 的字符集，因此不支持 Unicode 。

定义
单引号字符串

'我是字符串';

要表达一个单引号自身，需在它的前面加个反斜线（\）来转义。要表达一个反斜线自身，则用两个反斜线（\\）。其它任何方式的反斜线都会被当成反斜线本身：也就是说如果想使用其它转义序列例如 \r 或者 \n，并不代表任何特殊含义，就单纯是这两个字符本身。

不像双引号和 heredoc 语法结构，在单引号字符串中的变量和特殊字符的转义序列将不会被替换。

双引号字符串

"我是双引号字符串, 我可以使用$variable";

在双引号中, 变量名将被变量值所取代. 而在单引号中, 变量名称, 或者任何其他的文本都会不经修改而发送给浏览器.

转义字符


			\n    // 换行
			\'    // 单引号
			\"    // 双引号
			\\    // 斜杠

注意: 单引号字符串只支持\'和\\转义字符, 双引号字符串支持所有的转义字符.

heredoc字符串

echo <<<theEnd
	line 1
	line 2
	line 3
	theEnd

PHP4新增了heredoc语法(<<<): 通过一个用来结束字符串的结束标记, Heredoc语法允许指定长字符串. 如上所示的代码创建了一个3行的字符串并且回显它们. theEnd标记时非常模糊的, 它只需要保证不会出现在文本中.
