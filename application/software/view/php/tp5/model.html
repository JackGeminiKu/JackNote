ThinkPHP是一个快速、简单的基于MVC和面向对象的轻量级PHP开发框架，遵循Apache2开源协议发布.

TP5基于PHP5.4设计(完美支持PHP7).

TP5安装&设置
TP5运行环境要求
PHP >= 5.4.0 （完美支持PHP7）
