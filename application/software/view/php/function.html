string getType(mixed var);

指定形参类型


function demo(array $options) {
    var_dump($options);
}

从PHP5开始, 可以指定形参的类型, 当调用函数时, 如果实参的类型与形参的类型不符, 那么就会产生一个致命级别的错误(Catchable fatal error).
1) 从PHP 5.0.0开始就支持用类名限制参数类型，限定以后，传入的参数必须是这个类的实例。
2) PHP 5.1.0开始增加了对array的支持，传递的参数必须是数组。
3) PHP 5.4.0开始支持又支持了callable这个伪类型，传递的参数必须是一个可以回调的函数。
4) PHP 7.0.0开始支持bool/float/int/string，传递的参数必须是布尔值/浮点数/整数/字符串
