常用算法
排序 - 冒泡法

// C语言版本
void bubble(int a[], int n)
{
    int i, j, t, temp;
    for (i = 1; i <= n - 1; i++) {
        t = n - i;
        for (j = 0; j <= t - 1; j++)
            if (a[j] > a[j + 1]) {
                temp = a[j];
                a[j] = a[j + 1];
                a[j + 1] = temp;
            }
    }
    return;
}

将相邻两个数a[0]与a[1]比较, 按要求将这两个数排好序; 再将a[1]与a[2]比较, ...; 依次处理, 直到将最后两个数比较并处理完毕. 这时最大(或最小)的数已换到最后一个位置. 这是第一轮的比较和处理. 每进行一轮, 就把剩下的树中的最大(或最小)的一个移到最后位置, 共进行若干轮.

排序 - 选择法

void selectsort(int a[], int n)
{
    int i, j, p, temp;
    for (i = 0; i < n - 1; i++) {
        p = i;
        for (j = i + 1; j < n; j++)
            if (a[j] < a[p]) p = j;
        if (p != i) {
            temp = a[p];
            a[p] = a[i];
            a[i] = temp;
        }
    }
    return;
}

从所有元素中选择一个最小元素放在a[0](即让最小元素a[p]与a[0]交换), 作为第一轮; 第二轮是从a[1]开始到最后的各元素中选择一个最小元素, 放在a[1]中; ...以下依次类推.

选择法比冒泡法的交换次数少, 效率较高

查找 - 折半法

#define n 15
void binary_search(int a[], int n, int x)
{
    int mid, top, bot, i, find;
    top = 0;
    bot = n - 1;
    find = 0;
    do {
        mid = ( top + bot ) / 2;
        if(x == a[mid]) {
            printf("find:%3d, it is a [%2d]",x,mid);
            find = 1;
        } else if(x<a[mid]) {
            bot = mid - 1;
        } else if(x>a[mid]) {
            top = mid + 1;
        }
    } while((top <= bot) && (find == 0));
    if(find == 0) printf("%3d not been found.",x);
}
